# Qoffa 🥗

## 📌 Description
Qoffa is a mobile application developed as a Bachelor's Degree final project (Licence 3 Informatique, Algeria).

The application aims to reduce food waste by allowing restaurants to sell their unsold food at discounted prices to nearby consumers.

## 🌍 Problem
Food waste is a major environmental and economic issue in Algeria.  
Restaurants often throw away unsold food at the end of the day.

## 💡 Solution
Qoffa connects restaurants and users through a mobile platform where:

- Restaurants can publish surplus food offers
- Users can browse available meals nearby
- Users can reserve food directly through the app

## 🛠 Technologies Used

- **Flutter**
- **Dart**
- **Firebase Authentication**
- **Cloud Firestore**
- **Firebase Storage**

## 📱 Features

- User authentication (Sign up / Login)
- Restaurant account management
- Food offer publishing
- Reservation system
- Real-time database updates
- Image upload for food offers

## 🏗 Architecture
The application follows a client-based mobile architecture using Flutter for the frontend and Firebase as Backend-as-a-Service (BaaS).
